This is the PDF documents folder for the QA API.

INSTRUCTIONS:
1. Place your PDF files in this folder
2. Restart the API application to load new documents
3. Use descriptive filenames as they appear in API responses

SUPPORTED FILES:
- Only PDF files with .pdf extension
- Multiple PDFs are supported
- File names should be descriptive (e.g., "research_paper_2024.pdf")

EXAMPLES:
- technical_manual.pdf
- company_report_q4_2024.pdf
- research_findings.pdf

After adding PDFs, restart the application with:
python api.py

The documents will be loaded into the vector database at startup.
